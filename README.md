# Blender3DWorld

基于blender 引擎下的3D世界，内容包括但不局限于3D打印，3D建模，3D游戏，3D地图，VR，AR

[Spaceship Generator](https://github.com/a1studmuffin/SpaceshipGenerator):随机生成3D飞船模型的脚本
